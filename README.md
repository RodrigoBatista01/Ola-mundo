## Oque pode fazer ?
### Funcionalidade 1:
> Mostrar as suas conversas recentes do whatsapp.
### Funcionalidade 2:
> Seleciona umas das conversas para poder enviar mensagem.
### Funcionalidade 3:
> Mostra as ultimas mensagens na conversa selecionada.
